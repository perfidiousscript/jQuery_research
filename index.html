Details

This page utilizes two instances of the jQuery function '.offset()'. When placed after a selector .offset() will return the distance (in number of pixels) that the element is from the top and from the left of the page. In this case the selected element is the grey box, and the location values are being passed into the locator box in the top left of the page. If .offset() is called with values inside of it the element will be moved to the position of those values. This script calls two random values into offset everytime the box is clicked, thus randomizing the box's position.
